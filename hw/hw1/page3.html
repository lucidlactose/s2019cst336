Without going into to much detail, C was a very popular language and was very versatile. In terms of paradigms, C is an imperative, procedural language. Internally it was similar to other languages at the time such as Fortran, but was more easily accessible. Procedural languages for a long time looked and functioned very similarly to C, but over time, interpreted languages began to be more popular and more diverse. Although the paradigm is retained, the layout and syntax of these languages deviated from C. Funnily enough, some interpreted languages such as Python use C or C# to interpret.
