What are some examples of C-like languages?

There are many languages out there that can be described as C-like. If the screenshot above looks familiar, your programming language may be C-like, even if the syntax looks a bit off. The more popular term would be C-family, although it does not directly have to branch off of C to be called either. If you would like an more fully made list, there is a page on wikipedia here.

  • Fortran
  • ALGOL
  • Java
  • Julia
  • Javascript
  • PHP
  • All C derivatives such as C++
  • B
  • C
  • D
  • F
  • J
